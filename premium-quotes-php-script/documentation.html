Requirements

  1. PHP 53 or higher
  2. MySQLi
  3. Rewrite Engine (htaccess)

Preparing

  1. Open your FTP and connect to your server
  2. Upload the scripts content
  3. Make sure you uploaded the .htaccess file so that seo url's work
  4. CHMOD 777 the core/database/connect.php file
  5. CHMOD 777 or 755 all the folders inside user_data

Installing

  1. Access http://yourdomain.com/install.php with your browser
  2. Enter the required details
  3. Hit enter and if there are no errors delete the install.php
  4. You can now login with your admin account( Username: admin ; Password: admin )
